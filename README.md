# 🚚 FleetTraq

FleetTraq is a web-based fleet management system built with ReactJS and Firebase, designed to help businesses monitor and manage their vehicle operations with real-time tracking and streamlined logistics.

## 🌟 Features

- 🔍 Real-time GPS vehicle tracking with Leaflet
- 🛠 Maintenance scheduling and alerts
- 🧑‍✈️ Driver profiles and performance metrics
- 📍 Trip history and route visualization
- 🛢 Fuel usage monitoring
- 📊 Admin dashboard for fleet overview
- 🔔 Notifications for key updates

## 🧰 Tech Stack

- **Frontend**: ReactJS, CSS
- **Backend**: Firebase (Auth, Firestore, Hosting, Cloud Functions)
- **Maps**: Leaflet.js (OpenStreetMap)
- **Deployment**: Vercel (#fleettraq.vercel.app)
